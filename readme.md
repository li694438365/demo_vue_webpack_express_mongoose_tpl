# vue + webpack + express + mongoose + cmd.js

### 1.模板说明

* 前端vue2.0
* dev和build：webpack
* 后台express提供API
* 数据库用mongoose操作。
* cmd.js
```
1.一键上传本地数据库到服务器，服务器自动部署。
2.一键下载本地数据库到项目根目录
3.一键上传项目根目录下的数据库到本地数据库
4.一键提交代码(里面有 下载数据库到根目录)
```

* 前后端分开监测，webpack进行代理后台的服务。
